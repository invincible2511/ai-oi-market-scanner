## Designed for Extension

The architecture allows easy expansion:

* Historical OI storage
* Backtesting modules
* Statistical signal evaluation
* ML-based classification layers
* Multi-broker support

---

## Recommended Extension Order

1. Data persistence layer
2. Signal quality analytics
3. Performance tracking
4. Confidence calibration
5. Visualization dashboard

Each can be added without altering core scanner logic.

---

## What Should Not Be Added

* Automatic trade execution
* Capital management
* Leverage optimization
* Risk exposure automation

Those belong in separate systems.
